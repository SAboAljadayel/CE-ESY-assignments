- تنفيذ المخزن الدائري (Circular Buffer) بلغة C

-وصف المشروع:

يهدف هذا المشروع إلى تنفيذ **المخزن الدائري (Circular Buffer)** باستخدام لغة C بدون استخدام أي هياكل بيانات جاهزة.

المخزن الدائري هو مصفوفة تعمل بشكل دائري، حيث عند الوصول إلى نهاية المصفوفة يتم الرجوع إلى بدايتها.



- الخصائص :

* تهيئة المخزن (Initialization)
* عملية الكتابة (Write)
* عملية القراءة (Read)
* التعامل مع الحالات الخاصة:

  * امتلاء المخزن (Buffer Overflow)
  * فراغ المخزن (Buffer Underflow)



-فكرة العمل:

يعتمد المخزن الدائري على ثلاث متغيرات رئيسية:

* head → مؤشر القراءة
* tail → مؤشر الكتابة
* count → عدد العناصر داخل المخزن

يتم استخدام عملية **modulo (%)** لضمان الحركة الدائرية داخل المصفوفة.



- طريقة عمل البرنامج:

1. إدخال اسم المستخدم
2. إضافة النص  "ESY-CE"  إلى الاسم
3. تخزين كل حرف داخل المخزن الدائري
4. قراءة البيانات من المخزن
5. طباعة الناتج على الشاشة
6. التأكد أن المخزن أصبح فارغًا بعد القراءة



- الاختبار:

تم اختبار البرنامج في حالتين:

* عندما يكون حجم المخزن أصغر من عدد الأحرف → يحدث Overflow
* عندما يكون حجم المخزن أكبر من عدد الأحرف → يعمل بشكل صحيح
